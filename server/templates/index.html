APRM & Thermal Solver for RBWR

A lightweight, borderless desktop overlay for the Realistic Boiling Water Reactor (RBWR) Roblox game. Calculate reactor power (APRM) and steam flow requirements in real-time.

Key Features

Designed to sit seamlessly on top of Roblox for quick telemetry monitoring.

Borderless HUD Mode

A borderless window that stays on top of Roblox. You can easily adjust the window opacity from 30% to 100% in settings.

Auto Screen Scanning (OCR)

Automatically reads target demand values directly from the game using OCR when you press the hotkey (default: F7).

Dynamic Power Solver

Calculates exact thermal requirements by dynamically factoring in power usage from feedwater, condenser, and recirculation pumps.
